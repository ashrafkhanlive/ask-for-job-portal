🚀 Ask For Job Portal – Online Recruitment Management System

A full-featured, high-performance Online Job Portal System designed to streamline the recruitment process for job seekers, recruiters, and administrators.

This project provides a secure, scalable, and user-friendly platform that connects employers with potential candidates efficiently.

📌 Project Overview

Ask For Job Portal is a web-based E-Recruitment Management System that allows:

👨‍🎓 Job Seekers to create profiles, search jobs, and apply online

🏢 Recruiters to post jobs and manage applicants

👨‍💼 Administrators to monitor and manage the entire system

The system ensures secure data handling, centralized database management, and smooth communication between users.

🎯 Objectives

Streamline the hiring process

Centralized candidate data management

Improve candidate screening & filtering

Enhance employer decision-making

Provide secure and scalable architecture

Support multi-user access with role-based control

📂 Installation & Setup

Clone the repository

git clone https://github.com/your-username/ask-for-job-portal.git

Create Virtual Environment

python -m venv env

Activate Environment

env\Scripts\activate

Install Dependencies

pip install -r requirements.txt

Configure Database (MySQL)

Run Migrations

python manage.py makemigrations
python manage.py migrate

Run Server

python manage.py runserver
